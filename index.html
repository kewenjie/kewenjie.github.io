Redis

redis参考命令 http://redisdoc.com/index.html

redis的安装

  • tar zxvf 命令解压
  • 进入到解压后的目录，make一下，可能会报错
    • yum install -y gcc
    • make MALLOC=libc
    • yum install tcl
  • 启动服务就可以了。

redis的数据结构

1.String

redis的key的string是如何存储的，sdsDr8默认，默认’\0’结束，以hello为例

len 5
alloc 5
flag 001
buf[] buf[] ‘h’ ‘e’ ‘l’ ‘l’ ‘o’

使用场景

  • ip限制，短信验证，nameSpace key的命名需要注意。

2.list

使用场景：

  • 实现消息队列

  • 实现栈 后进先出 lpush lpop 实现队列：先进先出 lpush rpop
